What is javascript?

javascript is a high-level , interpreted programming language. It executes line by line that's why this is called singles threaded language. It will do run time assignments.

What is a variable?

A variable is a container for a value, like a number we might use in a sum or a string that we might use as a part of a sentence.

They are 3 types..

  • Var
  • Let
  • Const

  1. Declaration

    It refers to the process of introducing a variable to the JS interpreter

    It involves specifying the variable name using a keyword like 'var', 'let' and 'const'

    Ex:

    • var x;
    • let y;
    • const z;

  2. Initialization

    It is the process of giving an value to a variable at the time of declaration or later in the program

    Variable can be declared without being initialized. In such cases, they hold Undefined value until a value is assign to them.

    EX:

    var x = 5; Declaration & Initialization

    let y; Declaration

    y=6; Initialization

    Const = "Hello" (const variables must be initialized when declared)

  3. Assignment

    It involves updating the value of an already declared variable

    Ex:

    var x = 5;

    x = 10;

  4. Hoisting

    Hoisting means that variable declarations are hoisted to the top of their scope during the compilation phase, while the assignments (Initializations) remains in place.

What is var key word in javascript?

The var key word is used to declare variables. It was the original way to declare variables in JS before the introduction of "let" and "const" in ECMAScript(ES6)

var x = 5;

Variable declare with var are function-scoped or globally scoped, which means they are accessible within the function or globally if declare outside of any function.

One important characterstic of var-decalared variable is that they have "Hoisting" behavior. This means that variable declarations are hoisted to the top of their scope during the compilation phase
